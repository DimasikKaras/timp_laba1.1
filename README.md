# Диспетчерская ПБ

SPA на React для управления объектами пожарной безопасности, персоналом и журналом событий.

## Доступные команды

### `npm install`

Устанавливает зависимости проекта.

### `npm run api`

Запускает `json-server` c данными из `/home/runner/work/timp_laba1.1/timp_laba1.1/db.json` на порту `5000`.

### `npm start`

Запускает React-приложение в режиме разработки на `http://localhost:3000`.

### `npm test`

Запускает тесты.

### `npm run build`

Собирает production-версию приложения.
